ansitelnet

Retro-faithful BBS terminal connector for Linux — single self-contained binary, no install required. Full 16-colour ANSI art, ZModem file transfer, built-in server book and live BBS directory.

Linux Python → single binary Telnet · ZModem · CP437 MIT licence
Download Binary GitHub Demos Docs

Install

# download, make executable, add to PATH
curl -L https://github.com/plastart/ansitelnet/releases/latest/download/ansitelnet-linux-x86_64 \
     -o ~/.local/bin/ansitelnet
chmod +x ~/.local/bin/ansitelnet

For ZModem file transfers you also need lrzsz: sudo apt install lrzsz

Features

▩
ANSI & CP437

Block/box-drawing characters and 16-colour IBM palette via OSC 4 — art renders exactly as on the original hardware.

▲
ZModem Transfer

Auto-detected downloads, F12 upload picker, batch transfers, binary Telnet mode — powered by lrzsz.

☰
Server Book

Persistent local list of BBSs with name, host, port and colour mode. Add, edit, delete with keyboard shortcuts.

♦
BBS Directory

Live download from telnetbbsguide.com — browse hundreds of active boards, filter by name, connect or save with one key.

▮
Width Cap

-w 80 centres the BBS in 80 columns with │ borders on wide terminals. NAWS-aware.

▶
Session Recording

Record sessions as asciinema casts (-r file.cast) and replay or edit them with the built-in player (--play).

Usage

Direct connection

ansitelnet <host> [port]            # Telnet, 16 colours
ansitelnet bbs.example.net 23
ansitelnet bbs.example.net 23 -8    # 8-colour mode
ansitelnet bbs.example.net 23 -n    # netcat mode (no IAC)
ansitelnet bbs.example.net 23 -w 80 # cap at 80 columns

Recording & playback

ansitelnet bbs.example.net 23 -r session.cast  # record
ansitelnet --play session.cast                  # replay / edit

Background

What is a BBS?

A Bulletin Board System (BBS) is a computer server that users dial (or telnet) into to exchange messages, play door games, browse files, and transfer software. BBSs flourished from the late 1970s through the 1990s and formed the backbone of pre-internet online culture. Today's surviving systems run classics like Mystic BBS, Synchronet, and Renegade.

ANSI art & CP437

BBSs use ANSI escape codes combined with IBM Code Page 437 — the original PC character set with block-drawing characters (░▒▓█), box-drawing lines (┌┐└┘│─), and symbols never in ASCII. ansitelnet handles CP437→UTF-8 conversion transparently and sets the 16-colour IBM palette via OSC 4.

ZModem

ZModem (1986, Chuck Forsberg) is the crash-recoverable file-transfer protocol that replaced XModem/YModem on BBSs. ansitelnet detects the \x18B trigger frame mid-stream and hands the socket to rz/sz, showing a live progress dialog.
